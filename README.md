# AWS ECR Scanner

A Python-based auditing tool that scans Amazon Elastic Container Registry (ECR) to identify empty repositories across your AWS account. Helps DevOps teams maintain clean container registries, reduce costs, and improve infrastructure governance.

## Features
- Automated scanning of all ECR repositories
- Identifies repositories with zero container images
- Exports detailed reports for cleanup decisions
- Secure credential management via environment variables
- Supports pagination for accounts with 100+ repositories

## Use Cases
- Cost optimization by identifying unused repositories
- Infrastructure cleanup and maintenance
- Compliance auditing for container registry governance
- Pre-migration assessments
